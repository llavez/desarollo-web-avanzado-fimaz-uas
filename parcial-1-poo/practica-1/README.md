Objetivo de la practica 
Implementar una clase en php que represente a un usuario

Descripcion de la clase 
La clase usuario cuenta con dos atributos privados nombre y correo 
Metodos usados:
Constructor
Getters (getNombre)
Setters (setNombre)

Intrucciones de ejecucion
Dentro de la carpeta "practica-1" crear los archivos index.php y Usuario.php
Ejecutar el archivo index.php en un servidor local
